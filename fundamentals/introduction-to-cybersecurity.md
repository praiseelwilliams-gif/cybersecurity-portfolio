#Introduction to Cybersecurity – Notes
Overview

##This note summarizes key foundational concepts in cybersecurity, including types of data, the McCumber Cube framework, and common indicators of phishing attacks.

Types of Data
1. Traditional Data
Transactional data (e.g., banking transactions)
Intellectual data (e.g., research, designs)
Financial data (e.g., account balances, payment records)
2. Internet of Things (IoT) and Big Data
IoT refers to a network of physical devices connected to the internet
These devices collect and exchange data in real time
This increases the volume of data and expands the attack surface
McCumber Cube Framework

The McCumber Cube is a three-dimensional model used to evaluate and design information security systems.

1. Security Principles (CIA Triad)
Confidentiality
Prevents unauthorized access to sensitive information
Examples: encryption, multi-factor authentication (MFA), identity verification
Integrity
Ensures data remains accurate and unaltered
Examples: hash functions, checksums
Availability
Ensures systems and data are accessible when needed
Examples: system maintenance, backups, redundancy
2. States of Data
Data in Use (Processing)
Data actively being processed by applications
Example: secure application handling
Data at Rest (Storage)
Data stored in databases or storage systems
Example: database encryption
Data in Transit (Transmission)
Data moving across networks
Example: HTTPS, secure communication protocols
3. Security Controls
Technical Controls
Tools and technologies used to protect systems
Examples: firewalls, encryption, antivirus
Administrative Controls
Policies and procedures guiding security practices
Examples: security policies, staff training
Physical Controls
Physical measures to protect infrastructure
Examples: CCTV, access control systems
Phishing Awareness

Phishing is a common cyberattack method used to trick users into revealing sensitive information.

Key Indicators of a Phishing Email:
Suspicious or misspelled sender email address
Unusual or misleading URLs
Poor spelling and grammar
Low-quality or inconsistent graphics
Urgent or threatening language (e.g., “act now”)
Key Takeaway

Cybersecurity involves protecting data across:

Different types of data
All states of data
Multiple layers of security controls

Understanding frameworks like the McCumber Cube helps in approaching security problems in a structured and comprehensive way.
